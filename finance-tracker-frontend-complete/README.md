# Personal Finance Tracker Frontend

Complete React frontend for the Spring Boot Personal Finance Tracker backend.

## Features

- React 18 + Vite
- Redux Toolkit
- Material UI
- Axios API integration
- JWT authentication
- Login/Register
- Protected routes
- Responsive desktop/tablet/mobile layout
- Mobile drawer navigation
- Hamburger menu
- Dark mode / light mode toggle
- Dashboard with charts
- Transaction CRUD
- Budget CRUD
- Search and filter
- Loading states
- Toast notifications

## Run

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:5173
```

## Backend

Expected backend URL:

```text
http://localhost:8080/api
```

Create `.env` from `.env.example` if required.
