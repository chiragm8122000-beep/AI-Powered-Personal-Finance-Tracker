# Personal Finance Tracker - Frontend

Responsive React application built using React 18, Redux Toolkit, Material UI, and Chart.js.

## Tech Stack
- React 18
- Vite
- Redux Toolkit
- Material UI
- Axios
- React Router
- Chart.js

## Features
- JWT Login/Register
- Dashboard
- Transaction CRUD
- Budget CRUD
- Search & Filter
- Charts
- Responsive UI
- Mobile Drawer Navigation
- Hamburger Menu
- Dark Mode
- Toast Notifications
- Protected Routes

## Run

```bash
npm install
npm run dev
```

Frontend URL:
http://localhost:5173

Backend URL:
http://localhost:8080/api
